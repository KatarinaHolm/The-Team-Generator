# The Team Generator

## About the Project
The idea for the app emerged when our teacher asked us to come up with creative ways to divide the class into groups during our workshops. 
We had recently started working with object-oriented programming, and the project became a perfect opportunity to practice creating classes and structuring a program into multiple modules.
The result was a console application in C# with features for adding and removing people from the class list, choosing how many people each group should consist of, and randomly assigning participants into different groups.

## Features
- Add/remove people
- Set group size
- Random grouping

## How to Run
1. Clone the repository:
   git clone https://github.com/your-username/The-Team-Generator.git
   
3. Navigate to the project folder:
   cd The-Team-Generator
  
4. Open the project in Visual Studio or run it using the .NET CLI:
   Using .NET CLI:
   dotnet run

4.The list of persons is currently hardcoded. To modify it, go to the Student file.
